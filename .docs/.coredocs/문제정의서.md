- TagaTutor 문제 정의서

- 1. 프로젝트 개요
- 프로젝트 명: TagaTutor (타가튜터)
- 핵심 목표: 일본어 텍스트를 입력받아, 컬러노트 식의 리스트로 관리하고, 선택 시 한컴타자처럼 연습할 수 있는 환경 제공.
- 컴퓨터 웹 : 로마자 입력 / 핸드폰 앱 : 플릭 타입 
- 배경: AndTrans의 텍스트 처리/UI 경험을 기반으로 일본어 학습/타자 연습에 최적화된 앱 개발.

- 일본어 원문 텍스트 입력 기능.
